Age of Adventure

April - June 2023

A web-based video game that incorporates object-oriented-programming and DOM manipulation to follow a storyline dependent on user input.

HTML5 JavaScript CSS3

Objective

  • Create an interactive, web-based game that allows users to make decisions to influence upcoming events and explore multiple narrative pathways
  • Demonstrate and improve proficiency in JavaScript, DOM manipulation, and UI/UX through a creative project with minimal restrictions and support

Process

  • Developed global state variables and dynamic updates to facilitate consistent, responsive gameplay using JavaScript
  • Structured the HTML and CSS layout to optimize UI design
  • Designed and mapped multiple story branches, including the entire narrative flow and all possible decisions and their outcomes
  • Implemented replay functionality, allowing multiple replays with fully reset state from various points in the storyline

Results

  • A choose-your-own-adventure game that combines creativity and programming knowledge to deliver an interactive storytelling experience that is unique to each user
  • Exceeded expectations of project, going above and beyond to produce a fun experience
